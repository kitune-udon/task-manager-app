# テスト観点 自動テスト対応表

## 1. 目的

`テスト観点一覧.md` の観点IDが、どのテストソースで検証されているかを追跡できるようにするための対応表。

テスト実行ログからも追跡できるよう、対象テストの `@DisplayName` または Playwright の `test(...)` 名にも観点IDを含める。
