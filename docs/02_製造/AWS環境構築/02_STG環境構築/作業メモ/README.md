# STG環境構築 作業メモ

staging 環境構築 作業手順書に沿って、章単位で作業状況を記録する。

## 作成ルール

- 手順書の章ごとに 1 ファイル作成する
- 章に着手した時点でメモを作成または更新する
- 実施結果、確認事項、保留事項を残す
- 手順書順に進めるが、順序差異が出た場合は理由も記録する
