# AWS予算上限・停止方針・削除方針候補メモ

## 目的

- 初回 AWS 公開における月額予算の上限を決める
- 想定外の課金が出たときに、どの順番で停止・調査・削除するかを明確にする
- MVP 段階で過剰な運用を避けつつ、請求事故を防ぐ

## 前提

- リージョンは `ap-northeast-1` 東京で確定している
- 初回公開は MVP スコープに限定する
- 構成は `RDS -> Elastic Beanstalk -> S3 / CloudFront` を前提とする
- 初回は単一の本番環境を中心に運用し、検証用の一時環境は必要時のみ作成する

## 方針決定で重視すること

- 想定外の高額請求を避ける
- 本番環境をむやみに止めない
- 一時検証環境は長く残さない
- 後から見て判断しやすいルールにする

## 予算候補

### 案A: 月額上限 10,000 円

#### 利点

- MVP 段階としては保守的で、異常な増加に早く気づきやすい
- 余計な構成追加を抑制しやすい

#### 注意点

- 一時的に構成を増やしたときに、早めに警告が出やすい

### 案B: 月額上限 15,000 円

#### 利点

- MVP 本番 + 軽い検証作業が重なっても、やや余裕を持って見られる
- 誤検知的な警告を減らしやすい

#### 注意点

- コスト増に気づくタイミングが少し遅くなる

## 推奨案

- 月額上限は 10,000 円を推奨する

## 推奨理由

- 初回 MVP 公開では、まず「請求事故を防ぐ」ことを優先したい
- 本番環境を小さく始める前提なら、監視の感度は高めのほうが扱いやすい
- もし構成追加で頻繁に警告が出るなら、その時点で 15,000 円へ見直せばよい

## 予算アラート案

- 50% 到達: 5,000 円で通知
- 80% 到達: 8,000 円で通知
- 100% 到達: 10,000 円で通知
- 予測超過: 月末予測が 10,000 円を超える時点で通知

## 監視の推奨

- AWS Budgets で月額コスト予算を 1 件作成する
- AWS Cost Anomaly Detection を有効化する
- 通知先は最低 1 つのメールアドレスを設定する

## 停止方針

### 基本方針

- 本番環境は、異常請求があっても即停止しない
- まず原因を確認し、停止候補は「一時環境 -> 非本番構成 -> 本番縮退」の順で判断する

### 停止順序

1. 一時検証用の Elastic Beanstalk 環境や不要 EC2 相当リソースがあれば停止または削除する
2. 不要な検証用 RDS や S3 バケットがあれば削除候補として切り出す
3. 本番以外の不要リソースを整理しても収まらない場合のみ、本番構成の縮退を検討する
4. 本番停止は、サービス継続より請求リスクが大きいと判断した場合に限る

### 本番縮退の考え方

- まずはインスタンスタイプ、台数、ログ出力量などの見直しを優先する
- DB を削除するのではなく、必要なら snapshot を取得したうえで縮退判断する

## 削除方針

### 一時環境

- 検証用に作成した一時環境は、利用終了後 24 時間以内に削除する
- 一時バケット、一時 DB、一時 Elastic Beanstalk 環境を残しっぱなしにしない

### 本番関連

- 本番 RDS を削除する場合は、必ず最終 snapshot を取得する
- 本番 S3 バケットは、中身と配信経路を確認してから削除する
- CloudFront は、参照先の S3 や DNS 影響を確認してから削除する

### ログと証跡

- 削除前に、対象リソース名、削除理由、実施日時をメモへ残す
- 予算超過時の対応も、誰が何を止めたかを記録する

## 今回の判断候補

- 月額上限: 10,000 円
- アラート閾値: 50% / 80% / 100% / 予測超過
- 停止方針: 本番即停止はしない。一時環境から順に止める
- 削除方針: 一時環境は 24 時間以内削除。本番 DB 削除前は snapshot 必須

## 最終決定

- 2026年4月8日、月額上限は 10,000 円で確定した
- アラート閾値は 50% / 80% / 100% / 予測超過で確定した
- 停止方針は「本番即停止はせず、一時環境から順に止める」で確定した
- 削除方針は「一時環境は 24 時間以内削除、本番 DB 削除前は snapshot 必須」で確定した

## 補足

- 初回運用で警告が過剰に出る場合は、月額上限を 15,000 円へ見直す余地を残す
- ただし、初回公開時点では請求事故防止を優先し、まずは 10,000 円で開始する
