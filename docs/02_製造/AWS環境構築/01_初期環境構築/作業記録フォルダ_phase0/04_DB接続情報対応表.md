# DB接続情報対応表

## 起動方式

- 方式: Docker Compose
- 定義ファイル: `backend/docker-compose.yml`
- サービス名: `db`
- コンテナ名: `task-db`
- イメージ: `postgres:15`

## 接続情報

| 項目 | 値 | 定義場所 |
| --- | --- | --- |
| DB名 | `taskdb` | `backend/docker-compose.yml` |
| ユーザー | `postgres` | `backend/docker-compose.yml` |
| パスワード | `postgres` | `backend/docker-compose.yml` |
| ホスト | `localhost` | Spring 設定ファイル |
| ポート | `5432` | `backend/docker-compose.yml` / Spring 設定 |

## backend 設定との対応

| Spring キー | 値 | 定義ファイル |
| --- | --- | --- |
| `spring.datasource.url` | `jdbc:postgresql://localhost:5432/taskdb` | `application.yml`, `application-dev.yml` |
| `spring.datasource.username` | `postgres` | `application.yml`, `application-dev.yml` |
| `spring.datasource.password` | `postgres` | `application.yml`, `application-dev.yml` |

## 起動/停止コマンド

起動:

```bash
docker compose -f backend/docker-compose.yml up -d db
```

状態確認:

```bash
docker compose -f backend/docker-compose.yml ps
```

停止:

```bash
docker compose -f backend/docker-compose.yml stop db
```
