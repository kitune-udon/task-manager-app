# Phase0 / Phase1 対応状況チェックリスト

## 目的

- AWS デプロイ準備に関する Phase0 / Phase1 の進捗を一覧化する
- 対応済み / 一部対応 / 未対応を明確にする
- 次に着手すべきタスクを迷わず判断できるようにする

## ステータス定義

- `対応済み`: 手順書の目的を満たす成果物または実装・確認が完了している
- `一部対応`: 実質的な作業は進んでいるが、専用成果物や最終確認が未了
- `未対応`: まだ着手していない、または目的を満たす成果がない

### Phase0 の残り
