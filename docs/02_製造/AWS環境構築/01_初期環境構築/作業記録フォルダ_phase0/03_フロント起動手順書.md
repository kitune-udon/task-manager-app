# フロント起動手順書

## 前提

- Node.js と npm が利用できること
- backend が `http://localhost:8080` で利用可能であること
- 作業ディレクトリがリポジトリ直下であること

## 使用技術

- React 19
- TypeScript 5.9
- Vite 8
- Axios

## 基本手順

### 1. 依存関係を確認する

```bash
cd frontend
npm install
```

補足:

- `frontend/package-lock.json` が存在するため、npm 前提の構成

### 2. 開発サーバーを起動する

```bash
cd frontend
npm run dev
```

今回確認結果:

- `npm run dev -- --host 127.0.0.1` で起動確認
- Vite が `http://127.0.0.1:5173/` を表示

### 3. 本番ビルドを作成する

```bash
cd frontend
npm run build
```

今回確認結果:

- `vite build` 成功
- `dist/` 生成を確認

## 起動確認ポイント

- ログイン画面が表示されること
- Network タブ上で backend の API 呼び出し先が `http://localhost:8080` になっていること
- build 後に `frontend/dist/index.html` と `frontend/dist/assets/*` が生成されること

## 既知注意点

- `.env.local` / `.env.production` は未作成
- API Base URL は現状コード固定
- backend 側の CORS も `http://localhost:5173` 固定
